Politique de Confidentialité

Dernière mise à jour : 17 janvier 2026

Lyam Apps accorde une grande importance à la protection de vos données personnelles. Cette politique décrit comment nous collectons, utilisons et protégeons vos informations.

1. Responsable du Traitement

Lyam Apps
Email : contact@lyamapps.com

2. Données Collectées

2.1. Données fournies par vous

  • Informations de compte : nom, email, téléphone
  • Contenu publié : photos, descriptions, messages
  • Informations de paiement : traitées par Stripe

2.2. Données collectées automatiquement

  • Données techniques : type d'appareil, version OS, identifiants
  • Données d'utilisation : pages consultées, actions effectuées
  • Données de localisation : avec votre consentement

3. Finalités du Traitement

Vos données sont utilisées pour :

  • Fournir et améliorer le service
  • Gérer votre compte
  • Traiter les transactions
  • Envoyer des notifications importantes
  • Personnaliser votre expérience
  • Assurer la sécurité
  • Respecter nos obligations légales

4. Base Légale

Le traitement de vos données repose sur :

  • L'exécution du contrat (fourniture du service)
  • Votre consentement (notifications marketing)
  • Nos intérêts légitimes (amélioration du service, sécurité)
  • Nos obligations légales

5. Partage des Données

Vos données peuvent être partagées avec :

  • Stripe : traitement des paiements
  • Supabase : hébergement des données
  • Apple/Google : gestion des abonnements
  • Autorités : sur demande légale
Nous ne vendons jamais vos données personnelles.

6. Transferts Internationaux

Vos données peuvent être transférées hors UE vers des prestataires offrant des garanties appropriées (clauses contractuelles types, certification Privacy Shield).

7. Durée de Conservation

  • Données de compte : durée de vie du compte + 3 ans
  • Données de transaction : 10 ans (obligations comptables)
  • Logs de connexion : 1 an
  • Messages : 5 ans après la dernière activité

8. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : obtenir une copie de vos données
  • Rectification : corriger des données inexactes
  • Effacement : demander la suppression
  • Portabilité : recevoir vos données dans un format structuré
  • Opposition : vous opposer à certains traitements
  • Limitation : limiter le traitement

Pour exercer vos droits : contact@lyamapps.com

9. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données :

  • Chiffrement des données en transit (HTTPS/TLS)
  • Authentification sécurisée
  • Accès restreint aux données
  • Surveillance et audits réguliers

10. Cookies et Technologies Similaires

L'application utilise des technologies de suivi pour :

  • Assurer le fonctionnement du service
  • Analyser l'utilisation (analytics)
  • Personnaliser l'expérience

11. Mineurs

L'application n'est pas destinée aux moins de 13 ans. Nous ne collectons pas sciemment de données de mineurs sans consentement parental.

12. Modifications

Cette politique peut être modifiée. Les changements significatifs vous seront notifiés.

Contact

Pour toute question sur vos données :

contact@lyamapps.com
